General Information

Provides support for interacting with HBASE from Streams.

Tuples can be added to HBASE via the HBASEPut operator (including with a checkAndPut condition) or incremented with the HBASEIncrement operator. Tuples can be retrieved with the HBASEGet operator. The HBASETableScan operator can output all tuples, or all tuples in a particular row range. The HBASEDelete operator enables tuples to be delted from HBASE.

Configuration uses the same hbase-site.xml that HBASE itself does.
