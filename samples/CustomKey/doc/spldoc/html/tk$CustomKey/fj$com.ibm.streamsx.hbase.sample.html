This page documents the properties of a set of native functions that may be invoked from SPL. The properties are used to map the SPL function interface to the native code and libraries that implement the function. The properties are defined using a Function Set that may declare a C++ header file, a CPP namespace, and the set of SPL interfaces for each of the native functions. If native functions require libraries, the library dependencies are also specified in the function set.
